# ECU — Engine Management

MegaSquirt-based engine management files for the Foxbody Mustang.

## Contents

| Folder / File | Description |
|---------------|-------------|
| `DashBoards/` | TunerStudio dashboard layouts (`.dash` files) |
| `Logs/` | Datalogs captured during tuning sessions (`.mlg`, `.csv`) |
| `Tunes/` | MegaSquirt tune files (`.msq`), custom INI overlays, and project settings |
| `Tools/portcheckv2b.exe` | Serial port check utility |
